# ERP-Assistant-AI
ERP Assistant AI — розумний ШІ-асистент для автоматизації обробки накладних. 🚀 📥 Telegram-черга: прийом фото накладних у фоні. 🧠 Smart-logic: розпізнавання акцій (7+2), перерахунок ящиків у КГ та пачок у ШТ. 📊 Агрегація: сумування товарів за кодом ЦБД перед введенням. 🤖 RPA: автоматичний друк даних в ERP через pyautogui.

# 🤖 ERP Assistant AI v1.9.5 — Розумний ШІ-Асистент для Накладних

Професійний інструмент на базі **Google Gemini AI**, що автоматизує повний цикл обробки накладних: від отримання фото в Telegram до автоматичного введення даних в ERP-систему.

## 🚀 Основні можливості

*   **🛰 Telegram-диспетчер**: Фонове завантаження фото накладних (поодиноких або альбомів). Автоматичне створення черги у вигляді структурованих папок.
*   **🧠 Бухгалтерський інтелект (v1.9.5)**:
    *   Розпізнавання акцій типу **(7+2)** або **(10+2)** з автоматичним перерахунком ціни.
    *   Розумна диференціація: перерахунок **ящиків у кілограми** (напр. 4.7кг) та збереження **пачок у штуках** (напр. 230г).
*   **📊 Розумна Агрегація**: Автоматичне підсумовування товарів з однаковим Кодом ЦБД (вага + вага, сума + сума) перед введенням.
*   **🤖 RPA Автоматизація**: Емуляція роботи оператора (друк даних у вікно ERP) з налаштованою затримкою 7 секунд для зручного перемикання вікон.
*   **📂 Система черги та статусів**: Візуальні індикатори стану накладної:
    *   📥 — Отримано (чекає на аналіз)
    *   ⌛ — Аналізується ШІ (фоновий процес)
    *   ✅ — Готово (дані розпізнано та збережено)
*   **🛠 Гнучке налаштування**: Зовнішній `config.py` дозволяє змінювати API-ключі та налаштування бази даних без перезбірки EXE.

## 🛠 Технологічний стек

*   **Мова**: Python 3.10+
*   **AI**: Google Generative AI (Gemini 1.5 Flash)
*   **GUI**: PyQt6 (багатопотокова архітектура)
*   **Database**: Firebird SQL (через fdb)
*   **Automation**: PyAutoGUI
*   **Messenger**: pyTelegramBotAPI

## 📦 Як користуватися

1.  Надішліть фото накладної боту в Telegram.
2.  Запустіть додаток на ПК. Накладна з'явиться у списку зліва.
3.  Натисніть **"Аналізувати"**. ШІ підготує таблицю з урахуванням акцій та фасовок.
4.  Перевірте дані та натисніть **"Ввести в ERP"**.
5.  Програма автоматично перенесе оброблену папку в `archive`.
